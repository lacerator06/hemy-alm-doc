# HEMY Application Lifecycle Management

By Leonard Estrada  
April 28, 2026

Welcome to the HEMY ALM documentation. This site contains the standards, rules, and guidelines that all Power Apps developers must follow when working across the HEMY environments.

## Environments

| Environment | Purpose |
|---|---|
| **HEMY EXPLORATION** | Feasibility testing and proof of concept |
| **HEMY DEV** | Rebuild validated concepts for delivery |
| **HEMY UAT** | Testing and sign-off before production |
| **HEMY PROD** | Live production environment |

## Key Principle

All developers share the same environments. There is no dedicated environment per developer. These rules exist to prevent conflicts, lost work, and deployment failures.
